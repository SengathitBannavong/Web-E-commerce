# Web-E-commerce
This is repo to build Web E-commerce for subject web technologies and e-Services (IT4409) with 4 contribute

## Server Run
### Require
- Node JS
### Build
```bash
  npm install
```
to build and install module and library
### Run
normal run
```bash
  npm run start
```
deverlop run
```bash
  npm run dev
```
**remember to init .env file**
